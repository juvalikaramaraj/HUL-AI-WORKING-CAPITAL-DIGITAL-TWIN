# HUL AI Working Capital Digital Twin

## Project Overview
This project is an AI-powered Working Capital Digital Twin developed to analyze, simulate, and forecast key working capital metrics.

## Objectives
- Forecast DIO, DSO, DPO
- Monitor Cash Conversion Cycle (CCC)
- Improve working capital efficiency
- Support financial decision-making

## Tools Used
- Microsoft Excel
- Power BI
- VBA
- HTML Dashboard
- Python

## Key Features
- AI-based forecasting
- Interactive dashboard
- Scenario analysis
- KPI tracking
- Working capital optimization
